Bug Tracker

I created a productive and easy-to-use Bug Tracker using Python and JSON. It allows developers to use their console to report bugs based on their severity with the developer's name, type of bug, priority of the bug, bug description, status of the bug, and time a developer found a bug. It makes finding and fixing bugs much easier using JSON files assigned a bug ID so developers can manage debugging.
